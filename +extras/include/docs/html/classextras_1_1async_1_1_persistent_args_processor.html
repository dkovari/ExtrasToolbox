Detailed Description

template<class PersistentArgType = cmex::mxArrayGroup>
class extras::async::PersistentArgsProcessor< PersistentArgType >

Async Processor with optional Persistent Arguments When the thread operates on a task the stored persistent arguments are concatenated to the task array group.

Usage: To add arguments call: obj.setPersistentArgs(nrhs,prhs); this will add prhs[...] to any future task arguments (created by pushTask)

This class is abstract, you need to implement ProcessTask()
