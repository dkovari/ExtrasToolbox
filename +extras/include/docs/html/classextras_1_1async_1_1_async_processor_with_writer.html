virtual bool extras::async::AsyncProcessorWithWriter::ProcessLoopCore ( )
inlineprotectedvirtual

Core method called by ProcessLoop() to handle tasks.

This function is responsible for calling ProcessNextTask() it should return a bool specifying if there are more tasks to process

Unless you know what you are doing, it is not a good idea to redefine this method (even though it is virtual) The method has been redefined here to include the ability to write the results to a file
